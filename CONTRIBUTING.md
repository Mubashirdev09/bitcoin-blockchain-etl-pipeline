\# Contributing — Bitcoin Blockchain ETL Pipeline



Thanks for taking the time to look at this project!



This is a personal portfolio project but contributions,

suggestions and feedback are always welcome.



\---



\## Ways to Contribute



\### Report a Bug

If something is broken or not working as expected:

1\. go to the \*\*Issues\*\* tab on GitHub

2\. click \*\*"New issue"\*\*

3\. describe what happened and what you expected

4\. include any error messages you saw



\### Suggest an Improvement

Have an idea to make the pipeline better?

1\. open an \*\*Issue\*\* with the label "enhancement"

2\. describe your idea clearly

3\. explain why it would be useful



\### Submit a Pull Request

Want to contribute code directly?



1\. fork the repository

2\. create a new branch:

```bash

&#x20;  git checkout -b feature/your-feature-name

```

3\. make your changes

4\. push to your fork:

```bash

&#x20;  git push origin feature/your-feature-name

```

5\. open a Pull Request on GitHub

\- keep Python code clean and readable

\- add comments explaining WHY not just what

\- test your changes before submitting
